# "The Box" 

S'ha simulat una empresa anomenada The Box. L'empresa és una entitat bancaria que te una arquitectura per tal d'administrar els seus clients.

# Instruccions

 Al directori actual, ens trobem un executable Bash (terminal) que ens permet iniciar l'entorn amb l'arquitectura. Simplement fent doble click al executable "init.bat", s'iniciaritzarà el sistema. Com a prerrequisit cal tenir instal·lat Vagrant a l'ordinador.

 Vagrant, web d'instal·lació -> https://www.vagrantup.com/downloads.html

 Un cop executat l' "init.bat" i desplegada l'arquitectura, sóls cal obrir el navegador i anar al localhost (127.0.0.1:8081) i s'obrirà un portal web amb els serveis que ofereix l'empresa.
